Portfolio

On this page, you can view the various programming projects that I have created over the years. You can also find music that I have written for some of these projects. Each of the projects are sorted by chronological order.

C# Projects

The projects below were developed using Unity and C#.

La Resistance (March 2020)

Created for the University of Derby game jam in March 2020, La Resistance is a two player arcade game where you must collect as much stolen treasure as possible.

Escape to the Afterlife (August 2020)

My first released Unity project as well as my first solo game jam, Escape to the Afterlife sees you as the spirit of the Pharaoh Necherophes platforming his way through his own tomb to get to the afterlife.

Phantom Pathfinding Test (November 2021)

This project was created to test and understand Unity’s in-built pathfinding system. The project uses graphics from The Legend of Zelda: Phantom Hourglass and mimics the Temple of the Ocean King dungeon.

Stop-Distance Test VR Application (April 2022)

Created for my final year project at university, this application is a VR implementation of the stop-distance test, an experiment designed to measure an individuals’ personal space.

Cellborn (June 2022)

Developed during my third year game development module at university, Cellborn is a top-down fantasy shooter where you play as an elf named Elvin, questing to escape the Evil Wizard’s tower.

Project Avalanche (August 2022)

This game started life as an application testing the Photon Pun 2 framework for multiplayer games in Unity.

VortexOS (Started September 2022)

My first attempt at releasing a decently sized project, this horror game sees you going against strange entities to rid a hacker from your computer.

The Patience Test (March 2023)

The goal of The Patience Test is simple: Sit in a room for as long as possible!
